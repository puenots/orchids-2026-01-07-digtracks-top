# DigTracks Static Website

Static HTML/CSS/JS website for DigTracks music service.

## Structure

```
rails_static/
  ├── index.html          # Main landing page
  ├── verify.html         # Email verification page
  ├── css/
  │   └── styles.css      # Main stylesheet
  ├── js/
  │   ├── app.js          # Main application logic
  │   └── i18n.js         # Internationalization (EN/JA)
  └── users/
      ├── sign_in.html    # Login page
      └── password/
          └── new.html    # Password reset page
```

## Development

Open `rails_static/index.html` in a browser or serve with any static file server.
